-Thêm, sửa nhân viên

3.Quản lý khách hàng:
-Thêm, sửa, xóa thông tin khách hàng
4.Quản lý hóa đơn:
-Hóa đơn nhập:Thêm, sửa thông tin hóa đơn nhập; xem chi tiết hóa đơn nhập;Xuất report và PDF hóa đơn nhập.
